* Yêu Cầu Cài Đặt:
+ Máy phải có .NET Framework 3.7.2 trở lên
+ Máy phải có SQL Serve 2019 Express
* Lưu ý:
+ Nếu không có sẵn, có thể tìm các file cài đặt cần thiết tại mục DinhKem trùng với thư mục cài đặt phần mềm(Mặc định: "C:\Program Files (x86)\KVC BOOKSHOP\DinhKem" )
* Hướng dẫn cài đặt:
Chạy file KVCBOOKSHOP.msi
- Chọn thư mục cần cài đặt phần mềm
- Chọn Next -> Next -> Close
- Chạy file Create database.bat trong thư mục ứng dụng "C:\Program Files (x86)\KVC BOOKSHOP"
- Mở thư mục cài đặt, chạy file KVCBOOKSHOP.exe(Mặc định: "C:\Program Files (x86)\KVC BOOKSHOP\KVCBOOKSHOP.exe") hoặc mở shorcut tại desktop.
